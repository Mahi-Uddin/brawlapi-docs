# Get Club

This endpoint retrieves club profiles, with basic stats and member info.

## HTTP Request

`GET https://api.brawlapi.cf/v1/club?tag=TAG`

### URL Parameters

| Parameter | Description                          |
| --------- | ------------------------------------ |
| `TAG`     | The club tag of the club to retrieve |

## Response
